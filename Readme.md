# Alfred alternative

## Build instructions

You need to have rust/cargo installed as well as node/yarn.

```
$ cargo --version
 cargo 1.46.0 (149022b1d 2020-07-17)
$ rustc --version
 rustc 1.46.0 (04488afe3 2020-08-24)
$ node --version
 v15.12.0
$ yarn --version
 1.22.10
```

To build & run:

```
make run
```
